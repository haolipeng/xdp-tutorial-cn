** Basic03：使用 BPF 映射计数

所有三个作业的解决方案可以在以下文件中找到：

 * [[file:../basic04-pinning-maps/common_kern_user.h][common_kern_user.h]] 文件包含新的 =datarec= 结构定义。
 * [[file:../basic04-pinning-maps/xdp_prog_kern.c][xdp_prog_kern.c]] 文件包含新的 =xdp_stats_map= 映射定义和更新的 =xdp_stats_record_action= 函数。

请注意，为了在后续课程/作业中使用，代码已移动到以下文件：
[[file:../common/xdp_stats_kern_user.h][xdp_stats_kern_user.h]] 和
[[file:../common/xdp_stats_kern.h][xdp_stats_kern.h]]。因此，为了在后续 XDP 程序中使用
=xdp_stats_record_action= 函数，只需包含以下头文件：

#+begin_example c
#include "../common/xdp_stats_kern_user.h"
#include "../common/xdp_stats_kern.h"
#+end_example

对于用户空间应用程序，只需要前一个头文件。
